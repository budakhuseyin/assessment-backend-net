# Aras Contact Directory

Aras Kargo stajı kapsamında geliştirilen, mikroservis mimarisine sahip telefon rehberi ve raporlama uygulamasıdır. 

Sistem temel olarak kişilerin iletişim bilgilerinin tutulduğu bir rehber servisi ve asenkron çalışan bir raporlama servisinden oluşmaktadır.

## Teknolojiler ve Mimari
* **Framework:** .NET Core
* **Mimari:** Clean Architecture, CQRS
* **Veritabanı:** PostgreSQL, MongoDB
* **Mesajlaşma:** RabbitMQ (MassTransit)
